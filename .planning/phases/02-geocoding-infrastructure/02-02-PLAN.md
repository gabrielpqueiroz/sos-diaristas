**1. Create `src/app/api/dashboard/mapa/route.js`:**

Follow the exact pattern from relatorios/route.js. Export a `GET` function.

Two queries:
- Pins query: `SELECT id, lat, lng, status, address, scheduled_date FROM crm_orders WHERE lat IS NOT NULL AND status IN ('concluido', 'agendado', 'confirmado', 'diarista_atribuida', 'em_andamento') ORDER BY scheduled_date DESC`
- Counts query: `SELECT count(*) FILTER (WHERE lat IS NOT NULL) as with_coords, count(*) FILTER (WHERE lat IS NULL AND address IS NOT NULL) as without_coords FROM crm_orders`

Response shape:
```json
{
  "pins": [{ "id": "uuid", "lat": -25.54, "lng": -54.58, "status": "concluido", "address": "...", "scheduled_date": "..." }],
  "total_with_coords": 42,
  "total_without_coords": 8
}
```

Parse counts with `parseInt()`. Error handler returns `{ error: 'Erro ao buscar dados do mapa', detail: error.message }` with status 500. Include `console.error('Error fetching mapa data:', error.message, error.stack)`.

**2. Update `src/app/dashboard/_components/MapComponent.js`:**

Add `Marker` and `Popup` imports from `react-leaflet`. Accept `pins` prop with default `[]`. Render markers inside MapContainer after TileLayer:

```javascript
import { MapContainer, TileLayer, Marker, Popup } from 'react-leaflet'

export default function MapComponent({ pins = [] }) {
  return (
    <MapContainer center={FOZ_CENTER} zoom={DEFAULT_ZOOM} style={{ height: '100%', width: '100%' }} scrollWheelZoom={true}>
      <TileLayer ... />
      {pins.map((pin) => (
        <Marker key={pin.id} position={[pin.lat, pin.lng]}>
          <Popup>
            <span style={{ color: '#1a1a2e' }}>
              {pin.address || 'Endereco nao disponivel'}
            </span>
          </Popup>
        </Marker>
      ))}
    </MapContainer>
  )
}
```

Keep ALL existing code (icon fix, FOZ_CENTER, DEFAULT_ZOOM, TileLayer config). Only add: Marker/Popup imports, `pins` prop, and the `pins.map()` block. Do NOT add a dynamic key to MapContainer (prevents remount — see Research pitfall 5). Popup text uses dark color `#1a1a2e` because Leaflet popup has white background.
